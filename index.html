Color en codigo Hexadecimal

El color hexadecimal es un sistema de notación numérica utilizado para representar colores en la mayoría de las páginas web. Se basa en la combinación de tres valores numéricos: rojo, verde y azul.

Color en Keywords

Las keywords son un concepto básico e indispensable para varios procesos, por el poder que tienen para elevar el trabajo que haces como parte de tu estrategia de mercadotecnia digital.

Color en RGB

El término RGB se refiere al color que reproduce las mezclas de tres colores primarios. El espacio de color RGB recibe el nombre de estos tres colores: rojo, verde y azul (en inglés, red, green, blue).

Color en RGBa

RGBA son la siglas para red green blue alpha. Es el modelo de color RGB incluyendo el alpha channel (canal alfa). El alfa indica la opacidad y permite que una imagen se combine con otras.

Color en HSL

Este esta formado por las siglas de Hue (tono) que determina el espectro del color; Saturation (indica la intensidad de un tono concreto); y Lightness (brillo), haciendo la diferencia entre brillo y obscuridad.

Color en HSLa

El modelo de color HSLA utiliza tres valores para definir el color (tono, saturación y luminosidad) y un cuarto valor opcional para la transparencia (alfa). background-color: hsla(261, 46, 74, 70);
